The file virtual.c can be compiled by navigating to noramczl-asgmt4 and typing gcc -std=c99 virtual.c -o virtual

The file virtual.c runs with arguments such as: ./virtual filename [FIFO | LRU | OPT]

For example: 	./virtual input.txt FIFO	./virtual input.txt LRU	 	./virtual input.txt OPT
	   	./virtual input2.txt FIFO	./virtual input2.txt LRU	./virtual input2.txt OPT
